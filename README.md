# Intelligent-Traffic-Corridor-Using-the-P-Programming-Language
This software presents a Cyber-Physical System (CPS) for intelligent traffic corridor management, focusing on formal modeling and rigorous safety verification. Developed using the P programming language, the system controls two synchronized intersections via a green-wave strategy, dynamically adapting to vehicle flow, pedestrian requests, public transit priorities, emergency vehicle detection, and accident management. The entire control logic is implemented as communicating state machines and formally verified using the PChecker framework to guarantee that conflicting traffic movements never receive green signals simultaneously.To demonstrate practical applicability, the formally verified P controller is seamlessly integrated with a C++ visualization module responsible for real-time traffic animation. While the C++ layer handles the communication and graphical interface, all decision-making operations remain strictly within the verified P component, preserving the correctness guarantees provided by formal verification. Experimental results under thousands of nondeterministic schedules showed zero deadlocks, assertion violations, or safety failures, proving the scalability and reliability of this hybrid architecture.
